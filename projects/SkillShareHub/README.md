# SkillShare Hub

A peer learning platform for users to teach and learn new skills through live sessions, tutorials, and collaborative projects.

## Features
- Create and join live learning sessions or workshops
- Upload and browse video tutorials and resources
- Collaborative project spaces for group learning
- Rating and feedback system for instructors and content
- Discussion forums and Q&A for each skill/topic
- Personalized recommendations based on interests and activity

## Getting Started
1. Open `index.html` in your browser to view the project.
2. All code is in `app.js` and styles in `style.css`.

## Contribution
- Follow the repository structure and guidelines in `CONTRIBUTING.md`.
- Add your project to `projects.json` (not `index.html`).
- Ensure responsiveness and accessibility.
