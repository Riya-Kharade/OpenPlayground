# PetPal

Connects pet owners for playdates and pet-sitting.

## Features
- Pet profiles (add/view your pet)
- Matchmaking (find playdates or pet-sitting matches)
- Messaging (simple chat simulation)

## Usage
1. Open `index.html` in your browser.
2. Add your pet's profile.
3. Find a match and start chatting!

## Tech Stack
- HTML, CSS, JavaScript (Frontend only)

## Notes
- No backend required.
- Chat and profiles are simulated for demo purposes.

## Contribution
- Add your project to `projects.json` (not `index.html`).
