# GreenCart

**Project Type:** Frontend (HTML/CSS/JS)
**Category:** Utility

## Description
GreenCart is an eco-friendly shopping assistant that suggests sustainable products and tracks purchases.

## Key Features
- Search and suggest sustainable products
- Add products to cart and track purchases
- Product history with local storage
- Responsive and accessible UI
- Carbon footprint tracking for purchases
