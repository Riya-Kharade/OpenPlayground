# Mood Tunes

**Project Type:** Frontend (HTML/CSS/JS)
**Category:** Utility

## Description
Mood Tunes detects your mood and generates a personalized playlist with an embedded player.

## Key Features
- Mood detection via user input
- Dynamic playlist generation
- Embedded music player (YouTube)
- Playlist history with local storage
- Responsive and accessible UI
